# Проект: Погода в терминале с requests

Учебный CLI-инструмент, который показывает текущую погоду для любого города, используя библиотеку `requests` и бесплатный API [Open-Meteo](https://open-meteo.com/).

## 📌 Задача
Продемонстрировать базовое применение `requests` для:
- отправки GET-запросов с параметрами
- обработки JSON-ответов
- корректной обработки ошибок и тайм-аутов

## 🛠 Требования

- Python 3.7+
- Библиотека `requests`

### Установка
```bash
pip install requests
